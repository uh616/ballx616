# Ball X Pit Evolution Viewer

Сайт‑справочник по эволюциям шаров и пассивных предметов из игры **BALL X PIT**.  
Позволяет быстро посмотреть, какие комбинации шаров и пассивок дают определённые улучшения.

Проект работает на **TypeScript + Vite + HTML/CSS**.
